link youtube : https://youtu.be/oomPT0toFXI

Pada percobaan ini dilakukan konfigurasi dasar jaringan menggunakan router, switch, dan dua PC pada Cisco Packet Tracer. Setiap PC berada pada subnet berbeda, sehingga router dikonfigurasi untuk memungkinkan komunikasi antar jaringan melalui pengaturan IP pada interface G0/0/0 dan G0/0/1, serta mengaktifkan routing IPv6. Switch juga dikonfigurasi dengan alamat IP VLAN 1 dan default gateway, sedangkan masing-masing PC diatur alamat IP dan gateway sesuai subnet. Setelah konfigurasi selesai, dilakukan pengujian konektivitas menggunakan perintah ping, di mana ping awal gagal sebelum konfigurasi dan berhasil setelah konfigurasi dilakukan dengan benar.
